# Módulo 2 - Introdução a Redes: Conectando computadores

## O que aprendi:
- Instalar o Packet Tracer
- Criar uma rede com 3 computadores
- Teste de conectividade
- DNS e NSlookup
- Cabos de conexão

## Testes Realizados:
- Nslookup

## Observações:
- Após fazer a instalação do Packet Tracer, criei uma rede com 3 computadores com IP fixo, e logo após criar a rede, fiz o teste de conectividade utilizando o comando ping via cmd.
- Descobri qual o papel do DNS quando acessamos um website, e descobri que ele que faz o mapeamento do endereço de IP e o nome do domínio.
- Utilizei o comando nslookup para obter informações sobre o servidor DNS do domínio que utilizei, como por exemplo: Endereço de IP e Servidor de domínio.
- Aprendi sobre a conexão direta de cabos, sobre a crimpagem do conector RJ45 e aprendi sobre o padrões de cores do cabo de rede direto, as conexões em que tive contato foram: T568A e T568B.
- Aprendi o motivo de usar o cabo cruzado, que serve para quando precisarmos conectar dois computadores diretamente, sem usar um dispositivo de rede como por exemplo um hub e descobri que algumas placas de rede são capazes de detectar quando conectamos um cabo errado, quando não temos um cabo cruzado, e que ela identifica e corrige as polaridades via software.
